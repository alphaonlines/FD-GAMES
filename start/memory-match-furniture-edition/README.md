# Memory Match: Furniture Edition

Card-flip memory game featuring furniture silhouettes, textures, and brand logos.

## Gameplay
- Present a grid of face-down tiles; tap to flip and reveal items.
- Match pairs (sofa/sofa, bed/bed, logo/logo, texture/texture).
- Include a Hard mode with larger grids (e.g., 6x6) for advanced players.

## Controls
- Tap to flip; quick replay/reset to keep sessions short.
- Large tiles and generous spacing for touch kiosks.

## Scoring
- Fewer moves and faster time yield higher scores.
- Track streaks for consecutive matches without misses.

## Content and branding
- Use FD product silhouettes, fabrics, and logos for tile art.
- Seasonal packs (holiday textures, outdoor sets) keep the game fresh.
