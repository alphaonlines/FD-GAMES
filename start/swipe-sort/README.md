# Swipe Sort: Bedroom vs Living Room

Sorting game where items appear one at a time and players swipe them into the correct room category.

## Gameplay
- Show one item at a time and prompt players to sort it into the right room (Bedroom, Living Room, optional Outdoor/Office/Kids).
- Encourage streaks by speeding up or adding trick items.

## Controls
- Large swipe zones (left/right or up/down) with supporting tap buttons for accessibility.
- Vibrant feedback for correct/incorrect placements.

## Scoring
- Accuracy plus a streak multiplier; streak resets on a miss.
- Optional time bonus for clearing a batch quickly.

## Content and branding
- Use real product images or silhouettes; label items with material or style hints.
- Rotate categories for special events (e.g., Outdoor vs Office).
