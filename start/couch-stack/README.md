# Couch Stack

Balance and placement game where players drag furniture silhouettes onto a moving platform and keep the stack from tipping.

## Gameplay
- Drag and drop large furniture pieces onto a moving platform.
- Maintain balance as the stack grows; avoid tipping over.
- Earn bonuses for precise placement and keeping the stack stable over time.

## Controls
- Touch-first drag-and-drop for large pieces.
- Simple reset/retry button for quick rounds in a kiosk setting.

## Scoring
- Time survived plus number of items placed.
- "Perfect placement" bonuses for centered drops.
- Optional streak multiplier for multiple stable placements in a row.
