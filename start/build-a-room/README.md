# Build-a-Room: Before/After

Tap-and-drag upgrade game where players improve an ugly room to raise its comfort score.

## Gameplay
- Start with a low-score room and objectives to improve comfort or style.
- Drag upgrades (paint, furniture, d√©cor) into place to transform the space.
- Show before/after scoring with clear objectives.

## Controls
- Tap to pick upgrades, drag to place; generous snap areas for kiosks.
- Quick undo/reset to try different layouts.

## Scoring
- Score based on completing objectives and time taken.
- Leaderboard: "Top Makeovers" to showcase best results.

## Content and branding
- Use FD catalog items for upgrades; align with promotions or new arrivals.
- Add themed challenges (small-space living, cozy bedroom, outdoor refresh).
