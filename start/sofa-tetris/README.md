# Sofa Tetris

Classic block-fitting puzzle using furniture silhouettes.

## Gameplay
- Drop furniture-shaped blocks (sectional, chair, ottoman) into a well.
- Rotate and position pieces to clear lines.
- Pace increases as players progress.

## Controls
- Tap rotate and drag to position; large left/right buttons for kiosks.
- Soft drop/hard drop options for advanced players.

## Scoring
- Points for each cleared line; bonus for combos and back-to-back clears.
- Track level progression to increase difficulty.

## Content and branding
- Furniture silhouettes styled after FD products.
- Seasonal block skins to keep visuals fresh.
