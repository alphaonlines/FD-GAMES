# FD Games Starter Kits

This folder packages design briefs and quick-start notes for each FD Games concept. Use these stubs to build touch-friendly kiosk experiences and track progress as we implement each title.

## How to use
- Pick a game below and open its README for gameplay, controls, scoring, and content needs.
- Add prototypes or assets inside the corresponding subfolder as you build.
- Update the checklist to reflect progress across the catalog.
