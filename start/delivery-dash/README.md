# Delivery Dash

Lane-based runner where a delivery truck avoids obstacles and collects reviews.

## Gameplay
- Control a delivery truck moving forward in lanes.
- Swipe to avoid potholes/traffic and collect "5-star reviews" or pickups.
- Increase speed over time to raise difficulty.

## Controls
- Swipe left/right for lanes; optional on-screen buttons for kiosks.
- Quick restart to keep sessions short.

## Scoring
- Distance traveled plus number of pickups.
- Bonus for flawless runs (no collisions) or collecting streaks.

## Content and branding
- Wrap the truck with FD/AlphaPulse branding.
- Add store-specific obstacles or pickup types to localize the experience.
