# Room Designer — Snap Fit

Quick drag-and-drop room designer built for touch. Drop furniture into the highlighted zones to finish a cozy living room as fast as possible.

## Goal
- Sofa, rug, coffee table, and lamp all placed in the correct outlines
- Fastest completion time with the fewest mistakes
- Earn the **Showroom Pro** badge for a perfect run (0 mistakes)

## How to play
1. Pick a style theme: Modern, Rustic, or Coastal.
2. Drag an item card from the tray onto the room canvas.
3. Get close to the outlined target—pieces will snap into place with a satisfying click.
4. Drop all four items to finish. Missed drops add a small penalty and return the piece to the tray.

## Scoring
- +10 points per correct snap
- -3 points for a miss, plus a mistake logged
- +25 bonus when the room is fully completed
- Perfect run bonus badge when you finish with 0 mistakes

## Tips
- Start the timer with your first drag; speed matters for leaderboard bragging rights.
- Use the restart button to retry quickly if you want the perfect badge.
- Themes are purely visual—play in whatever style matches your mood.
