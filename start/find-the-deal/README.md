# Find the Deal

Hidden-object tap game set in a crowded showroom image where players hunt for sale tags or featured products.

## Gameplay
- Present a detailed showroom scene and hide sale tags or featured items.
- Players tap the hidden target; optional hints appear after a few seconds.
- Rotate scenes weekly to highlight new promotions.

## Controls
- Large tap targets with forgiving hitboxes for kiosks.
- Hint button to flash the area or narrow the search zone.

## Scoring
- Fastest time to find the target; penalties for using hints.
- Fewer hints = higher score; show best times of the day.

## Content and branding
- Swap background images weekly with current promotions or store layouts.
- Highlight seasonal deals or clearance items to align with marketing.
